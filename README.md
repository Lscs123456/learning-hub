# 学习中心 · Learning Hub

秋招备战知识状态追踪工具，纯静态前端，数据存储在本地浏览器。

**在线访问：** https://lscs123456.github.io/learning-hub/

## 功能模块

### ⚡ LeetCode 刷题追踪
- 17 大题型分类，100 道精选题目
- 每题附 LeetCode 原题链接
- 按难度（简单 / 中等 / 困难）和掌握状态过滤

### 🧠 LLM 八股文知识地图
- 5 大模块，74 个核心知识点
- 内置 FSRS 间隔复习算法，自动安排今日复习计划

## 掌握度状态

| 状态 | 含义 |
|------|------|
| 未标记 | 未接触 |
| 初识 | 看懂合上忘 |
| 能讲 | 能完整复述 |
| 能用 | 能写 / 能应对追问 |
| 题破 / 融汇 | 内化成本能 |

点击任意题目或知识点右侧的状态标签即可循环切换。

## 数据说明

学习内容数据放在 `data/` 目录：

- `data/modules.json`：首页模块入口配置
- `data/leetcode.json`：LeetCode 题目分类和题目清单
- `data/llm.json`：LLM 知识点分组和优先级

进度数据仍保存在**本地浏览器 localStorage**，清除浏览器数据会导致进度丢失。各模块页面支持导出 JSON 备份。

## 技术栈

纯 HTML + CSS + JavaScript，无依赖，无构建步骤，通过 GitHub Pages 托管。
